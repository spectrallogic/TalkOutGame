# Free audio drop-in slots

The game synthesizes all its own sound (music, crowd, honk), so nothing here is
required — but real recordings sound better. Drop files in with these names and
the builders wire them up automatically on the next Build Everything.

## Crowd (The Open Mic) — `Assets/Audio/Crowd/`

| File | Used for |
|---|---|
| `laugh_big.wav` | full laugh wave (crowd won over) |
| `laugh_small.wav` | scattered chuckles |
| `cough.wav` | ambient cough in the silence |
| `murmur.wav` | uneasy room murmur |
| `boo.wav` | the room turning on you |

`.wav`, `.ogg`, or `.mp3` all work.

## Where to get free, commercially-safe audio

- **Pixabay Audio** (pixabay.com/sound-effects) — search "laugh track",
  "audience laughing", "crowd boo". Pixabay Content License: free for
  commercial use, no attribution required. Easiest source.
- **Freesound.org** — filter license to **CC0** only (Creative Commons 0).
  Requires a free account to download originals.
- **Kenney.nl** — CC0 game audio packs (UI clicks, impacts — good for the
  interactables later).
- **Sonniss GDC bundles** (sonniss.com/gameaudiogdc) — royalty-free for
  commercial games, huge packs.

Avoid: BBC Sound Effects archive (personal/education license only — NOT ok for
a commercial Steam release), YouTube rips (unlicensed).

Keep a note of the source URL + license for each file you add — Steam builds
should ship a credits list.
